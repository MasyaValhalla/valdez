# Discord Bot — Тикеты / Автопарк / АФК

## Установка

```bash
pip install -r requirements.txt
```

Создайте файл `.env` (по примеру `.env.example`):
```
TOKEN=ваш_токен_бота
```

## Запуск

```bash
python bot.py
```

## Все команды

### Тикеты
| Команда | Описание |
|---|---|
| `!тикет-панель` | Размещает панель с дропдауном для подачи заявок |
| `!тикет-настройка role <ID>` | Задать роль рекрутов |
| `!тикет-настройка results <ID>` | Канал для итогов заявок |
| `!тикет-настройка voice <ID>` | Войс-канал для обзвона |
| `!тикет-настройка category <ID>` | Категория для тикет-каналов |

### Автопарк
| Команда | Описание |
|---|---|
| `!автопарк` | Размещает панель автопарка с дропдауном |
| `!авто-добавить <название>` | Добавить авто |
| `!авто-удалить <ID>` | Удалить авто |
| `!авто-настройка channel <ID>` | Канал автопарка |

### АФК
| Команда | Описание |
|---|---|
| `!афк` | Размещает панель АФК с кнопкой |
| `!афк-настройка voice <ID>` | Войс-канал АФК (исключение из сбора) |
| `!афк-настройка channel <ID>` | Канал АФК-панели |

### Админ
| Команда | Описание |
|---|---|
| `!сбор` | Перемещает всех из войсов (кроме АФК) в ваш войс |
| `!рассылка @Роль сообщение` | Отправляет ЛС всем с указанной ролью |
| `!лог-настройка <категория> #канал` | Настроить канал логов (tickets/fleet/afk/vacation) |
| `!настройка` | Показать все текущие настройки |

## Быстрый старт

1. Запустите бота
2. `!настройка` — посмотрите что нужно настроить
3. `!тикет-настройка role 123456` — задайте роль рекрутов
4. `!тикет-настройка results 123456` — канал итогов
5. `!тикет-настройка voice 123456` — войс для обзвона
6. `!тикет-панель` — разместите панель тикетов
7. `!автопарк` — разместите панель автопарка
8. `!авто-добавить Sultan` — добавьте авто
9. `!афк` — разместите панель АФК
10. `!лог-настройка tickets #логи` — настройте логи
